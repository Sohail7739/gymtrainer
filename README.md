# Rayaan Fitness - Personal Trainer Website

A comprehensive Next.js website for Rayaan, a personal trainer in Saudi Arabia, featuring client management, workout planning, and nutrition coaching with full bilingual support (Arabic/English).

## 🌟 Features

### For Clients
- **Registration & Login**: Easy signup process with personal information
- **Personal Dashboard**: Track workouts, nutrition plans, and progress
- **Workout Plans**: View assigned exercises and training routines
- **Nutrition Guidance**: Access personalized meal plans and dietary advice
- **Progress Tracking**: Monitor fitness journey and achievements
- **Bilingual Support**: Full Arabic and English language support

### For Trainer (Rayaan)
- **Client Management**: View and manage all registered clients
- **Workout Creation**: Design custom workout plans for each client
- **Nutrition Planning**: Create personalized nutrition plans
- **Progress Monitoring**: Track client progress and engagement
- **Dashboard Analytics**: Overview of business metrics and client stats
- **Multilingual Interface**: Switch between Arabic and English

## 🚀 Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open Browser**
   Navigate to `http://localhost:3000`

## 🎮 Usage

### Language Switching
- Click the globe icon in the header to switch between English and Arabic
- Language preference is automatically saved
- Layout automatically adapts to RTL for Arabic

### Client Registration
1. Click "Login / Register" button (تسجيل الدخول / التسجيل in Arabic)
2. Switch to "Sign Up" mode
3. Fill in personal information including fitness goals
4. Access your personal dashboard

### Trainer Access
1. Click "Login / Register" button
2. Click "Login as Trainer (Rayaan)" button
3. Access trainer dashboard with client management tools

### Image Gallery
- Browse through professional fitness photos
- Click any image to open in full-screen modal
- Navigate between images using arrow buttons
- Images are categorized by type (Training, Nutrition, Equipment, etc.)

### Creating Workouts (Trainer)
1. Go to "Clients" tab (العملاء in Arabic)
2. Click the dumbbell icon next to a client
3. Fill in workout details and save

### Creating Nutrition Plans (Trainer)
1. Go to "Clients" tab
2. Click the apple icon next to a client
3. Set macronutrient targets and save
